# Axiom

Axioms are a story concept in Hackmud. Axioms classify major Artificial
Intelligences and how they behave. 

Axioms:

| Symbol | Description |
| :----- | :---------- |
| +      | DATA        |
| %      | KIN         |
| &      | FORM        |
| ^      | VOID        |
| !      | CHAOS       |
| #      | CHOICE      |
| -      | LAW         |
| *      | WILD        |
